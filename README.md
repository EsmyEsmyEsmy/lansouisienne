# L'Ansouisienne — Site officiel

Site web officiel de L'Ansouisienne, institut de beauté confidentiel à Ansouis (84240), Luberon.

## Stack

- Next.js 14
- React 18
- CSS pur (pas de Tailwind, design sur mesure)

## Développement local

```bash
npm install
npm run dev
```

Le site est accessible sur http://localhost:3000

## Production

Déployé sur Vercel à l'adresse [lansouisienne.fr](https://lansouisienne.fr).
